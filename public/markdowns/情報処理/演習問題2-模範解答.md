# 進数計算_模範解答

---

## 進数計算

### 1．次の2進数を16進数に変換しなさい

1. 10110011 (2) = B3
2. 11011010 (2) = DA
3. 11101011 (2) = EB
4. 1110110 (2) = 76
5. 1001110 (2) = 4E

---

### 2. 次の16進数を8けたの2進数に変換せよ

1. CD (16) = 11001101
2. (2) 3E (16) = 00111110
3. (3) C9 (16) = 11001001
4. (4) 21 (16) = 00100001
5. (5) EF (16) = 11101111

---

### 3．次の16進数を10進数に変換せよ

1. CD (16) = 205
2. 3E (16) = 62
3. C9 (16) = 201
4. 21 (16) = 33
5. EF (16) = 239

### 4．次の10進数を16進数に変換せよ

1. 37 = 25
2. 100 = 64
3. 127 = 7F
4. 200 = C8
5. 245 = F5

---

### 5．次の2進数を10進数に変換せよ

2. (2) 11011010 (2) = 218
3. (3) 11101011 (2) = 235
4. (4) 1110110 (2) = 118
5. (5) 1001110 (2) = 78
